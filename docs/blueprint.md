# **App Name**: OrgConnect

## Core Features:

- Authentication: Secure user authentication with login and admin panel access, differentiating between regular users and admin roles.
- Dashboard: Display the organization's logo, a title, and a brief description on the homepage.
- System List: Present a grid-style list of organizational systems on the homepage. Admins can add, edit, and delete systems via the admin panel.
- Online Status Indicators: Reflect the current user statuses using a green (online), yellow (idle), and red (offline) color-coded system.
- User Directory: Sidebar displays a filterable (by alphabet or department) directory of users synced from Active Directory, limited to 20 entries with a 'more' option to expand the list. Displayed directory entries include a photograph.
- User Profiles: Allow users to create and display a short bio on their profile.
- Messaging Integration: Integrate with Cisco Jabber (or similar) to launch a direct message to a user when their name is clicked.
- Admin Configuration: Enable admins to manage other admin users, customize site design/colors, and control login configurations.
- Login Reporting: Provide admins with reports on user login times, duration, and other relevant metrics.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey professionalism and trust.
- Background color: Light Indigo (#E8EAF6), a soft, desaturated tint of the primary, creates a clean and unobtrusive backdrop.
- Accent color: Teal (#009688) for interactive elements and highlights, providing a refreshing contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern and approachable feel.
- Use modern, flat icons to represent organizational systems and user actions.
- Implement a clean and structured layout with a prominent system list in the center and a user directory sidebar.
- Subtle transitions and animations on hover effects to provide a polished user experience.
